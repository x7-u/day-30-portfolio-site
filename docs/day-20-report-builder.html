Report Builder

Automated management reports from P&L plus KPIs
STACKPython / Flask / python-docx / DeepSeek FOCUSManagement reporting automation PORT:1020 TESTS21

About the project

Generate a fully formatted Word document plus Excel workbook of the management report from your P&L lines and KPI dashboard.

Local Flask app + CLI that turns this period's P&L numbers and KPI dashboard into a fully formatted Word document. DeepSeek V4 writes the period overview, variance commentary, key risks, and forward outlook; the engine assembles the cover, tables, embedded charts, and exports both .docx and .xlsx.

At a glance

Use case
Management-accounts commentary, board-pack period results
Inputs
P&L lines (current + prior) + KPI dashboard with direction flags
Outputs
Fully formatted Word doc + Excel workbook + three CSVs + three charts
Model
Single DeepSeek call (~$0.001 per pack)

How it works

  • Report schema. report_schema.py wraps the inputs in a typed ReportInput dataclass with PlLine and Kpi sub-records.
  • Report builder. report_builder.compute pulls the headline metrics by finding the standard P&L line names (revenue / gross profit / EBIT / PAT) in the user's input.
  • Narrative AI. narrative_ai.draft builds a pipe-delimited digest (every P&L line with current/prior/delta; every KPI with current/prior/format/direction/adverse flag) and asks DeepSeek V4 for...
  • Annual-report UI. Distinct from every prior day's palette.
  • Word + Excel + CSV exports. python-docx writes a fully formatted Word document: cover page, period overview, headline numbers table, P&L chart, variance commentary, KPI table + chart, margin walk chart, key...

AI integration

DeepSeek writes the narrative sections

Limitations

  • Two-period view only. Current + prior.
  • User supplies the data. No live ERP / accounting-system integration.
  • Single-template output. One template; no brand colours / custom logos.
  • GBP base only. Multi-currency consolidation is post-30.
  • AI proposes, does not opine. Every figure and narrative paragraph needs review by a finance director before publication.
  • No automatic distribution. Brief lists scheduled email distribution as post-30.

Fun facts

  • It turns a profit-and-loss and a set of KPIs into a fully formatted Word document and Excel workbook, ready for the board pack.
  • The fiddly formatting that eats an afternoon, done in moments.

Walk-through

  1. Home page (cold start) - Annual-report magazine aesthetic. Warm-white paper, deep navy masthead, gold accent rule, Georgia serif body.
  2. First data run: growth tech company - Lumen Cloud Ltd. Revenue up 18% Q-on-Q, margins steady, KPI dashboard mostly favourable.
  3. Second data run: declining manufacturer - Britannia Engineering plc. Revenue down 12%, margin compression, most KPIs adverse.
  4. Figures tab (what the Excel + Word exports embed) - Three charts on warm-white pages: P&L current-vs-prior grouped bar, KPI delta-pct bar (adverse red, favourable green),...
  5. AI narrative (the integration) - DeepSeek V4 reads the digest of P&L lines and KPIs and writes the four sections: period overview, variance commentary,...
