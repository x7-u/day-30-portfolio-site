About the project

Drop in receipts, payments, payroll and tax; the engine builds a rolling 13-week balance projection. DeepSeek V4 parses 'what if' prompts into deterministic shocks.

Local-only Flask web app and CLI for a 13-week rolling cash flow forecast with an AI-driven scenario engine. Drop in an XLSX, get a deterministic forecast plus a controller-voiced commentary on any "what if" you can describe in plain English.

At a glance

Model
DeepSeek V4 (deepseek-chat default)
Cost
around $0.0004 per analysis.

AI integration

DeepSeek parses scenarios + writes commentary

Limitations

  • GBP only in the MVP. Non-GBP metadata is rejected with a clear remediation message.
  • Period labels are opaque strings. April-26, 2026-04 and Q1 26 all work; they sort by first-seen order.
  • AI commentary is a draft. The fallback when the API errors is a stub card with the error reason; the rest of the analysis still runs.
  • Power BI integration not yet wired (Day 3 had it).
  • Run history caps at 100 with FIFO eviction.
  • Scenario taxonomy covers six shock types. M&A, hiring waves and capex spread are not yet supported.

Fun facts

  • Ask it 'what if revenue drops 20 percent for three weeks' in plain English and it rebuilds the whole 13-week cash forecast on the spot.
  • The AI only reads the numbers and picks the scenario; the maths is done in plain Python, so the forecast always reconciles.
