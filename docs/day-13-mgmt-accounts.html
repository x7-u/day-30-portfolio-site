About the project

Take a trial balance, produce the three primary statements with YoY comparison and AI commentary.

Local-only Flask web app and CLI that turns a single trial-balance .xlsx into a full monthly management-accounts pack: P&L by department, balance sheet (with balance check), indirect cash-flow statement, KPI dashboard with RAG bands, and a CFO-voice narrative drafted by DeepSeek V4. The maths is deterministic and reproducible; the AI never touches a number, only annotates the digest.

At a glance

Inputs
One trial-balance .xlsx for the monthly pack; an optional second one for year-over-year
Outputs
P&L by department + Balance + Indirect Cash Flow + KPI dashboard + AI commentary + YoY comparison
Model
DeepSeek V4 (deepseek-chat), one call per pack + one call per YoY narrative (deterministic fallback if offline)
Cost
around $0.0002 per CFO narrative; same again per YoY commentary

How it works

  • Trial-balance schema. tb_schema.py validates a two-sheet workbook (metadata + accounts).
  • P&L compute. pl_compute.py aggregates revenue / direct_cost / opex / depreciation / interest / tax into the standard headline lines (GP, EBITDA, EBIT, PBT, NI), computes margins safely, and...
  • Balance sheet compute. bs_compute.py aggregates assets / liabilities / equity, runs the balance check (assets = L + E within max(1.0, 0.1% of assets) tolerance), and computes working capital, current...
  • Indirect cash flow. cf_compute.py reconciles the indirect statement: NI + depreciation + interest + tax addbacks - working capital deltas - cash interest - cash tax = OCF.
  • KPI dashboard. kpi_compute.py produces 13 KPIs each with a comparator (prior month / budget / fixed target), delta, and RAG band.
  • AI commentator. ai_commentator.py builds a 250-token digest from the metrics (headline P&L, top departments, BS check, CF summary, top-5 variance lines) and asks DeepSeek V4 in JSON mode for...
