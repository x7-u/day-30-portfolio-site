Ratio Dashboard

Financial ratio analysis with AI commentary
STACKPython / Flask / openpyxl / Anthropic FOCUSFinancial statement analysis PORT:1001 TESTS36

About the project

Drop in financial statements; the engine computes liquidity, profitability and gearing ratios with RAG flags, and Claude writes the controller commentary.

Local-only Flask web app + CLI that computes 24 financial ratios from a company's two-period income statement and balance sheet, RAG-classifies each against sector-aware SME thresholds, and generates AI-grounded commentary from Claude. Self-contained Excel dashboard download. **Runs on your machine only - no third-party hosting.**

At a glance

Use case
Financial-statement analysis, SME ratio review with director commentary
Inputs
A company's two-period income statement and balance sheet (.xlsx); several files for multi-company comparison
Outputs
24 ratios with sector-aware RAG flags, AI commentary, and a self-contained Excel dashboard
Model
Claude Haiku 4.5 default, Sonnet 4.6 / Opus 4.7 selectable
Cost
around $0.0025 per run on Haiku 4.5, $0 with --no-ai

How it works

  • Ratio engine. ratios.py computes 24 ratios across liquidity, solvency, profitability and efficiency, then classifies each red / amber / green against dataclass thresholds.
  • Sector profiles. sectors.py overrides the thresholds for retail, manufacturing, services and SaaS so the RAG bands match the business model.
  • AI commentary. pipeline.py sends a compact pipe-delimited digest of the ratios to Claude, which writes a finance-director commentary; an offline --no-ai mode skips the call.
  • Excel output. excel_writer.py renders a dashboard with sortable numeric cells, RAG-coloured pills, an embedded chart and A4 print setup.

AI integration

Claude writes the management commentary

Limitations

  • Two-period analysis. Historical trend charting beyond two years is post-30.
  • No peer benchmarking. Sector averages from Companies House or public filings are post-30.
  • AI commentary is a draft. It is grounded in the computed ratios but should be reviewed before use.
  • Local-only. Loopback Flask dev server with no auth; production hosting is a later step.

Fun facts

  • It reads a set of financial statements and grades them with red, amber and green flags, so the health of a business jumps out before you have read a single number.
  • Day one of the run, and it already drafts its own management commentary in the voice of a financial controller.
