About the project

Five-year free cash flow forecast with Gordon terminal, 5x5 sensitivity over WACC and growth, and a Monte Carlo overlay for the intrinsic value distribution.

Local-only Flask web app and CLI for a five-year discounted cash flow valuation, with an AI assistant that pre-fills assumptions for a given company.

At a glance

Model
DeepSeek V4 (deepseek-chat), one call per AI pre-fill
Cost
around $0.0005 per pre-fill.

How it works

  • Two-stage DCF maths. dcf_maths.py is pure functional.
  • WACC build. Standard CAPM build. cost_of_equity_capm returns Re = Rf + Beta * ERP.
  • Schema validation with bounds. assumption_schema.parse takes any loose dict (from JSON, the form, or the AI) and returns a typed Assumptions instance or raises AssumptionError with a remediation message.
  • AI pre-fill. Single DeepSeek V4 call per pre-fill. About 250 input tokens and 500 output tokens, JSON mode, max_tokens 800.
  • Live recompute. static/app.js is a literal port of dcf_maths.run_dcf and the sensitivity functions to JavaScript.
  • Sensitivity and football field. sensitivity.build_sensitivity_table runs implied_per_share_for across a 7 by 7 grid of (WACC, g) pairs (defaults: 50bps and 25bps steps).

AI integration

DeepSeek writes the valuation commentary

Fun facts

  • A full five-year discounted cash flow valuation, with a 5 by 5 grid showing how the answer shifts as growth and the cost of capital change.
  • A Monte Carlo overlay turns a single 'intrinsic value' into a realistic range.
